---------------------------
1. SQL Injection (CWE-89) - Multiple locations
2. Cross-Site Scripting (CWE-79) - Templates and JavaScript
3. Cross-Site Request Forgery (CWE-352) - All forms
4. Insecure Direct Object References (CWE-639) - Todo operations
5. Path Traversal (CWE-22) - File download
6. OS Command Injection (CWE-78) - Admin panel
7. XML External Entity (CWE-611) - XML parsing
8. Server-Side Request Forgery (CWE-918) - URL fetching
9. Insecure Deserialization (CWE-502) - Pickle sessions
10. Hardcoded Credentials (CWE-798) - Config and Docker
11. Weak Cryptography (CWE-327) - MD5 password hashing
12. Information Disclosure (CWE-200) - Error messages
13. Missing Authentication (CWE-306) - API endpoints
14. Unrestricted File Upload (CWE-434) - No validation

DOCKER VULNERABILITIES
---------------------
- Python 3.6 base image (end-of-life)
- Running as root user
- Hardcoded secrets in ENV
- Privileged mode enabled
- Docker socket mounted
- No resource limits
- Overly permissive permissions (777)
- No health checks
- Using ADD instead of COPY
